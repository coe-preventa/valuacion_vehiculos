# Gemini Workspace

This file provides context and instructions for the Gemini AI assistant.

## Project Overview

This project is a car valuation application. It consists of a Python backend using FastAPI and a frontend.

## Key Technologies

- Python
- FastAPI
- Streamlit

## How to Run

1.  **Backend:**
    - Navigate to the `backend/api` directory.
    - Run `uvicorn main:app --reload`
2.  **Frontend:**
    - Navigate to the `frontend` directory.
    - Run `streamlit run app.py`
